How to get initial Node references (Windows version)

To be able to use Freenet you will now have to get initial node references, that is information about how to contact your first Freenet nodes. This is because Freenet is totally decentralized and every new node needs a starting point to get hooked up into the network. Your node will learn about other nodes over time though.

To keep Freenet decentralized we decided not to provide the same starting point for everybody. Which is why we push the task of finding an initial reference off to you. Solve it in the manner  you prefer, the more entropy in our network the better. We know that this sounds inconveniant, but it is the price you have to pay for decentral anonymous networking.

We will now list several possibilities on how to get initial Freenet references, pick the method which you prefer and save the gathered reference as seed.ref in your Freenet directory.
  • Get a node reference from a friend you trust. This is the most recommended method. Pick a reference file from somebody you know. On Windows these reference files are automatically created on installation (MyOwn.ref), so you just need to get that file from a friend and save it as seed.ref in your Freenet directory.
  • Other methods of getting references manually are over IRC (e.g. #freenet on irc.openprojects.net), mailing lists or public Freenet nodes. Be creative!
  • Save the seed of the day (shift-click to download) into your Freenet directory. This is the least recommended method as it is an untrusted reference.
To learn more about nodes and network joining you might want to visit the documentation section of http://freenetproject.org .
